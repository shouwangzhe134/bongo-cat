## 🌟 Core Highlights
- **Real-time Keyboard & Mouse Tracking** — Bongo Cat reacts to your inputs
- **Streaming Overlay Ready** — Perfect for OBS Studio, Streamlabs, Twitch, and YouTube
- **Custom Skins Support** — Load your favorite Bongo Cat variations
- **Chroma & Transparent Background** — Easy integration with game capture

## 🎮 Perfect For
- **osu! Players** — Classic Bongo Cat key tracker
- **Streamers** — Live keyboard visualization overlay
- **Desktop Pet Lovers** — Cute always-on companion
- **Content Creators** — Fun reactive animations

## 🚀 Key Features
- **Customizable Skins** — Support for community-made Bongo Cat themes
- **Low Resource Usage** — Runs smoothly in the background
- **Windowed & Overlay Mode** — Use as desktop pet or streaming widget
- **Configurable Settings** — Adjust sensitivity, animations, and position
- **Game Capture Friendly** — Works great with chroma key and transparent backgrounds

---
## 📖 How to Get Started
1. **Download** the latest version using the button above.
2. **Extract** the archive and run the application.
3. **Choose your skin** — Load default or custom Bongo Cat assets.
4. **Position & Configure** — Adjust size, transparency, and tracking options.
5. **Add to OBS** — Use Window Capture or Game Capture for streaming.
